## Deskripsi Project
Aplikasi Kasir Toko Sayur merupakan aplikasi berbasis Python yang dibuat untuk
membantu proses transaksi penjualan pada toko sayur.  
Aplikasi ini menyediakan fitur pengelolaan data barang, input transaksi,
perhitungan total harga, serta tampilan antarmuka grafis yang memudahkan pengguna.

Project ini dibuat sebagai **Project Akhir Mata Kuliah Pemrograman Visual**.

---

## Fitur Aplikasi
- Input data barang
- Proses transaksi penjualan
- Perhitungan total harga otomatis
- Antarmuka grafis (GUI)
- Penyimpanan dan pengolahan data transaksi

---

## Teknologi yang Digunakan
- Python
- PyQt / Tkinter (sesuaikan dengan library GUI yang digunakan)
- Visual Studio Code
- Git & GitHub
